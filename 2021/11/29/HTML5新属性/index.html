HTML新属性

HTML5

新增属性：

  1. 增加了audio 和video音频播放，抛弃了Flash

  2. 新增了canvas画布 （绘画 制作动画）

  3. 地理定位

  4. 增加了离线缓存

  5. 硬件加速

  6. Web Socket 全双工通信

  7. 增加了本地存储

  8. 新增了语义化标签

网页布局标签：

  • header 页首
  • nav 导航
  • aside 侧边栏
  • main 主体
  • section 区块
  • article 文章
  • footer 页尾

语义化标签

  1. mark 高亮显示

  2. details描述 summary 摘要 一般用于名词解释 summary为标题呈现

  3. meter 用来设置度量衡

    • value当前值

    • min最小值

    • max最大值

  4. progress

    • value当前值
    • min最小值
    • max最大值

  5. dialog对话框或者窗口

    在标签中加入open属性 &times；是❌号

  6. figure用来组合与元素 一般用来组合一张图的标题·图片·图片描述等

HTML多媒体

  • Audio 播放音乐或音频

  • 支持格式 .ogg .mp3 .wav

  • 属性：

    1. src文件路径

    2. autoplay自动播放

    3. loop循环

    4. controls控制条

    5. muted静音

    6. preload预加载 使用autoplay时，preload自动失效

  • video 加载视频

  • 支持格式 .mp4 .ogg .webm

  • 属性：

    1. src文件路径

    2. autoplay自动播放

    3. loop循环

    4. controls控制条

    5. muted静音

    6. preload预加载 使用autoplay时，preload自动失效

    7. width宽度

    8. height高度

    9. poster海报

  • embed 用来加载插件 嵌入内容

  • 属性

    1. src
    2. width
    3. height
    4. type类型
  • canvas画布 是一个容器元素
    1. 单独使用canvas没有什么意义，必须结合Javascript ，它的具体功能体现是通过javascript来体现的
    2. canvas的宽高不要通过css实现，而是直接通过width和height来实现

HTML新属性

  • contentEditable 将标签转换为可编辑状态

  • hidden 对元素进行隐藏，一般用来传值或当某个条件成立时，执行显示内容 默认值为hidden

  • data-*：用于存储页面或着应用程序的私有自定义数据，一般用于传值

  • multiple 规定输入域可选择多个内容 用于file/select

  • required 约束表单元素在提交前必须输入值 用于表单组件中，需要结合提交按钮


  • pattern 规定用于验证输入字段的模式

HTML新表单组件

  1. color颜色

  2. ⭐️email邮箱

  3. tel电话

  4. url网址

  5. number数字

  6. range范围

  7. search搜索

  10. datetime-local当地时间

  11. year年

  12. month月

  13. time时间

表单属性

  1. formaction修改action数据提交的地方 不是在form行操作

  2. formenctype修改表单请求的类型

  3. formmethod修改数据提交的方法

    • get是以字节为单位提交，只接受ASCII码，post是以字符为单位提交
    • get是明文方式，提交的数据会显示在地址栏中，而post是暗文方式提交
    • get在浏览器中回退是无害的，而post会再次提交请求
    • get会被浏览器缓存 而post不会，除非手动设置

  4. form设置表单元素属于哪个表单

  5. novalidate不验证

input属性

  1. autocomplete自动完成

    • 用来帮助用户输入，每一次输入的内容，浏览器是否保存输入的值，以备将来使用，有on/off 默认为on

    • 为了保护敏感数据，避免本地浏览器对它们不安全存储，一般需要关闭

  2. autofocus自动获焦

  3. step步长

  4. multiple多选

  5. pattern正则匹配

  6. placeholder输入提示

  7. required必须输入
