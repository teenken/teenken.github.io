Javascript基础--变量及运算

Javascript

  1. 三种引入方式：

    • 通过script标签写在任意地方，但是一般写在body后面,避免js堵塞和影响操作DOM
    • a标签的href中引入js文件
    • 用script标签的src属性引入外部js文件

  2. 变量名命名规则：

    • 变量名必须以字母、下划线 “_”、美元符号 “$” 开头，不能以数字开头
    • 变量可以包含字母、数字、下划线和美元符号
    • 不能使用 JavaScript 中的关键字做为变量名
    • 变量名不能有空格
    • 变量名对大小写敏感，比如：name 和 Name 就是两个完全不同的变量

  3. 通过 var 关键字来声明变量，变量类型：

    • number：浮点型，整数型都可以
    • string：字符串型，需要引号
    • boolean：true/false
    • array：数组型
    • object：对象型

  4. 值得注意的就是引号中如果是一个数字，它依然是 string 类型的。我们可以在控制台中通过 typeof 函数，来查看我们声明的变量是什么类型的

  5. num++ 和 ++num 的区别，前者是先赋值再自增，后者是先自增再赋值

  6. 运算符号

    • +
    • -
    • *
    • / 除 带小数 ex： 2/3=0.66666
    • ++
    • %取余
    • += x+=4 ————> x=x+4
    • -=
    • *=
    • /=
    • ===
    • !==
    • 大于>
    • 小于<
    • 大于等于>=
    • 小于等于<= 输出的值为true/false
    • &&
    • ||
    • ？： ex：c？x：y 代表判断c的值是true还是false true的话返回x，false的话返回y

  7. document.write :向body中写入一个内容

    Console.log在控制台输出一个内容

    typeof检查类型，输出结果是一个字符串

  8. 将js写入button的onclick属性中，写入a标签的href属性中

  9. script标签一旦引入外部文件后，在其内写入的代码不会起作用

  10. \表示转义符号 ， 加上引号就是字符串，不加就是变量

  11. 调用 XXX的YYY()方法就是：XXX.YYY()

  12. 将字符串转换为数字：

    parseInt 获得字符串中数字，可用parseInt（a，10）要求输出10进制

    parseFloat 获得字符串中小数

    Number

  13. 16进制0x开头 八进制0开头

  14. 数字—>布尔 除了0和NaN，其余都是true

  15. 字符串—>布尔， 除了空串，其余都是true

  16. Null和undefined转换为布尔都是false

  17. +可以实现字符串拼串效果，任何值和字符串做加法运算，都会先转换为字符串，然后再和字符串做拼串操作,可以利用这个来实现字符串的转换，将任意一个想要转换的加上空串就行
