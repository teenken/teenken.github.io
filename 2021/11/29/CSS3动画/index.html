CSS动画

CSS3动画

  1. 盒子的变形

    • transform:translate(x,y)

    • 或者把tranlate换成translateX （只在x方向做变换）translateY（只在y方向做变换）

    • transform:scale(x,y) x 方向变换x倍数 y方向变换y倍数x 缩放效果

    • scale（a）只带一个参数代表x，y都缩放a倍 还有scaleX scaleY

    • Transform:rotate(30deg)旋转30度

    • transform：skew(x-deg,y-deg)

    • transform:matrix(a,b,c,d,e,f)

      a c e

      b d f

      0 0 1

      a,d表示缩放（如果没有缩放，值设为1），b,c表示扭曲（如果没有扭曲，值为0）

      e,f表示位移（如果没有位移，值为0）

    • tranform-origin：（x，y）设置旋转基点

    • perspective 让子元素获得透视效果 主流浏览器都不支持，谷歌要加webkit，或者在长度后加单位

    • transform-style：flat/preserve-3d 规定如何在3D空间中呈现被嵌套的元素，必须与transform 属性一同使用 加在父元素标签内

    • rotateZ rotateX rotateY

CSS3过渡动画

  • transition-property

  • Transition-duration

  • Transition-timing-function

  • Transition-delay

  • Transition:可以使用属性：

    • 颜色：color background-color border-color outline-color

    • 位置：background-position left right top bottom

    • 长度：max-height min-height max-width min-width height width

      Border-width line-height text-indent vertical-align border-spacing

      Letter-spacing word-spacing

    • 数字：opacity visibility z-index font-weight zoom

    • 组合：text-shadow transform box-shadow clip

    • 渐变:gradient

  • Duration 动画持续时间：一般以秒或毫秒为单位s/ms

  • timing-function：动画函数

    • linear：匀速

    • ease：变速（慢快慢）

    • ease-in（由慢到快）

    • ease-out（由快到慢）

    • ease-in-out（慢开始，慢结束）

    • cubic-bezier(n,n,n,n):自行设定变速，n的值在0到1之间

  • delay动画延迟时间：s/ms为单位

@keyframes

关键帧动画：

  1. 设置关键帧：

  2. 实施动画：

    Animation-name 动画名 来自于@keyframes定义的动画名

    Animation-duration 持续时间

    Animation-timing-function 动画函数

    • linear：匀速（默认）
    • ease：变速（慢快慢）
    • ease-in（由慢到快）
    • ease-out（由快到慢）
    • ease-in-out（慢开始，慢结束）
    • cubic-bezier(n,n,n,n):自行设定变速，n的值在0到1之间

    Animation-delay 动画延迟时间

    Animation-iteration-count 动画循环播放的次数 number/infinite（无限循环） 默认1次

    Animation-direction 动画播放完后是否反向播放

    • normal不反向（默认值）
    • alternate 反向

    Animation-play-state 单独使用 动画播放或停止播放

    • paused：停止播放
    • running：播放（默认值）

    Animation: name duration timing-function iteration-count direction;
