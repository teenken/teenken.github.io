HTML基本标签

  1. 改变网址的代表图片：

    在head标签中使用link:favicon来实现

  2. 引入css样式三种方法：

    • 在heead标签中使用link:css
    • 在head标签中使用style标签进行包裹
    • 直接在语句中写入样式

  3. br用来换行，单标签

  4. hr用来生成一条水平线，单标签

  5. a标签实现跳转功能，中间写入提示文字，在其中写入title属性，表示鼠标悬停时的信息

  6. a标签的target属性中 _blank新建一个窗口, _self表示在当前窗口打开

  7. img中的title属性显示鼠标悬停信息

  8. span标签与div一样，但不会独占一行

  9. i和em标签可以使文字倾斜，em还起强调作用

  10. sub，sup分别设置下标和上标

  11. 表格属性中rowspan表示跨行，colspan表示跨列

  12. 表格分为：thead tbody tfoot tr表示一行 td表示一列

form表单

  1. 表单常用属性：

    name：表单名称

    action ：表单数据提交的地方

    method：提交方法 有 get和 post两种

  2. 表单元素：

    • input：输入，用type来具体定义：type:text/password/radio/checkbox/file/button/image/submit/reset

    • placeholder：提示文字：Username：Password：

    • value设置默认值

    • minlength 设置最少输入字符

    • maxlength

    • readonly只读

    • radio单选钮

    • checkbox复选框，后面加入选项

    • file文件上传

    • button普通按钮，通常用来调用脚本文件

    • image 图片按钮，用法与button一样

    • submit提交按钮 用来提交数据

    • reset 重置按钮 清空还原为原来状态

    • textarea 文本域 多行文本框 用于大批量内容常用属性 id name cols rows placeholder提示文字 minlength maxlength

    • select选项 与optipn配合添加选项， 可以通过multiple实现多选 用command 用size可以调整选择框大小

文字处理

  1. text-transform 字母大小写 capitalize每个词第一个字母大写 uppercase全部大写 lowercase转换成小写

  2. font-weight 加粗 100～900 700相当于bold 400相当于normal

  3. 伪类：表示元素的一种特殊状态

    • a:link 初始状态
    • a:visited 访问过后的状态
    • a:hover 鼠标悬停时状态
    • a:active 选择器用于选择活动链接
    • focus 获得焦点

  4. id选择器使用：[id=’’]{}

  5. Float:initial 表示选择默认值，即none

  6. 清除浮动有三种方法：

    clean:left/right/both/none 清除左/右/都/都不 添加空标签清除浮动

    overflow:hidden 清除浮动的父级 添加样式 超出部分隐藏 可以用来实现清除浮动

    ：after 添加尾元素 父元素:after{content=””;display:block;clear:both;}

  7. 行级标签没有宽高，块级标签有

  8. 定位：position

    • static默认值 没有定位
    • relative 相对定位 占用文档流 可以设置偏移，但原来占据的位置不能填充其他元素
    • absolute 脱离文档流，相对于body做偏移左上角（0，0）
    • absolute与relative配合使用，相对于relative（父级）做移动
    • fixed 脱离文档流，相对于（0，0）作位移
    • z-index一般用在absolute 上 调整显示顺序
